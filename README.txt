--------------------------------------------------------------------------------
------------------------------- OOHOO CUSTOM LIST ------------------------------
--------------------------------------------------------------------------------

--------------------------------------------------------------------------------
Description
--------------------------------------------------------------------------------

This block allows to generate a list of links to be displayed in the course.

--------------------------------------------------------------------------------
Prerequisites
--------------------------------------------------------------------------------

No prerequisites

--------------------------------------------------------------------------------
Installation
--------------------------------------------------------------------------------

1.	copy the folder renamed "custom_list" into the moodle/blocks folder.
2.	Login to Moodle with admin rights. 
3.	In the Site Administration block, click on Notifications. This will 
        setup the database tables for the plugin

You are done. Go into a course, turn editing on and add a the block Custom list.

--------------------------------------------------------------------------------
